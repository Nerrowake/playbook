# Nerrowake Playbook

The Nerrowake playbook defines shared engineering, design, documentation, and
repository standards for Nerrowake projects.

Nerrowake is a small, founder-led software studio that builds thoughtful
software, developer tools, interactive systems, community tools, and
game-related utilities. The goal of this playbook is to keep projects practical,
maintainable, and clear as they grow.

## What This Covers

- project vision and principles
- repository structure
- GitHub setup
- branching and Git workflow
- code style
- documentation
- testing
- releases
- security
- accessibility
- design systems
- definition of done

## Core Values

- Clarity over cleverness
- Maintainability over novelty
- Correctness before performance
- Documentation as part of the product
- Thoughtful design
- Practical engineering
- Long-term ownership

## Planned Project Context

Strata is planned as the flagship Nerrowake developer tool. It will be a
real-time staging telemetry dashboard for Laravel teams, focused on requests,
queries, N+1 issues, jobs, and scheduled tasks during staging QA.

This playbook prepares the foundation for that work. It does not contain Strata
application code.
