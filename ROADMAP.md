# Roadmap

This roadmap tracks planned improvements to the Nerrowake playbook.

## Now

- Keep baseline standards aligned with active Nerrowake repositories.
- Use the playbook as the reference for new repository setup.

## Next

- Add repository bootstrap checklists.
- Add example GitHub Actions workflows.
- Add release note examples for applications and developer tools.

## Later

- Add architecture decision record templates.
- Add deeper guidance for telemetry, privacy, and observability products.
